🎭 Yoya Modal 弹出框组件

完全透明的弹出框控制，内容由您自定义

📦 基础用法

最简单的弹出框使用方式，通过 content() 方法设置内容，footer() 方法设置底部按钮。

📝 带标题的弹出框

使用 title() 方法设置弹出框标题，标题区域会自动添加分隔线。

✅ 确认框

使用 vConfirm() 快速创建确认对话框，支持自定义确认/取消回调和按钮文本。

🎨 自定义内容弹出框

弹出框内容完全由您控制，可以嵌套任何组件，如表单、表格、卡片等。

🔒 不可关闭的弹出框

通过 closable(false) 和 maskClosable(false) 禁用关闭功能， 适用于需要强制用户操作的场景。

📏 自定义尺寸

使用 width() 方法设置弹出框宽度，支持任何 CSS 单位。
