# Icons 图标库

## 概述

Yoya.Basic 提供 72+ 个 SVG 图标，使用 PascalCase 命名，直接返回 SVG 元素。所有图标都支持主题适配（使用 `currentColor`），可以方便地通过 CSS 修改大小和颜色。

## 使用方式

### 导入图标

```javascript
// 从 @yoya/basic 导入图标
import { HomeIcon, UserIcon, SettingsIcon } from '@yoya/basic';

// 或者从本地导入
import { HomeIcon, UserIcon } from '../../yoya/index.js';
```

### 基础用法

```javascript
// 方式 1：直接插入 SVG
const app = document.getElementById('app');
app.appendChild(HomeIcon());

// 方式 2：作为子元素
import { div } from '@yoya/basic';

div(box => {
  box.child(HomeIcon());
  box.text('首页');
});

// 方式 3：在 JSX 风格中使用
div(box => {
  box.icon(HomeIcon());
});
```

### 自定义样式

```javascript
// 修改大小
SettingsIcon({ width: '32', height: '32' });

// 修改颜色（使用 CSS）
const icon = HomeIcon();
icon.style('color', '#ff0000');

// 修改描边宽度
SearchIcon({ 'stroke-width': '1.5' });

// 使用主题变量
const icon = BellIcon();
icon.style('color', 'var(--yoya-text-primary)');
```

### 在组件中使用

```javascript
// 在按钮中使用
import { vButton, SearchIcon } from '@yoya/basic';

vButton(btn => {
  btn.child(SearchIcon({ width: '16', height: '16' }));
  btn.text('搜索');
});

// 在菜单项中使用
import { vMenuItem, UserIcon } from '@yoya/basic';

vMenuItem(item => {
  item.icon(UserIcon());
  item.text('用户资料');
});

// 在卡片中使用
import { vCard, vCardHeader, StarIcon } from '@yoya/basic';

vCard(card => {
  card.vCardHeader(header => {
    header.text('收藏项');
    header.child(StarIcon());
  });
});
```

## 技术细节

### SVG 实现

所有图标都是纯 SVG 实现：

```javascript
// 图标函数内部实现
function createSvgIcon(viewBox = '0 0 24 24', props = {}) {
  const svg = document.createElementNS(SVG_NS, 'svg');
  svg.setAttribute('viewBox', viewBox);
  svg.setAttribute('fill', 'none');
  svg.setAttribute('stroke', 'currentColor');
  svg.setAttribute('stroke-width', '2');
  // ... 设置其他属性
  return svg;
}
```

### 主题适配

图标使用 `currentColor` 作为描边颜色，自动继承父元素的文字颜色：

```css
/* 亮色主题 */
:root {
  --yoya-text-primary: #333;
}

/* 暗色主题 */
[data-theme="dark"] {
  --yoya-text-primary: #fff;
}

/* 图标自动继承颜色 */
.icon {
  color: var(--yoya-text-primary);
}
```

### 尺寸规范

- **默认大小**: 24x24 (viewBox)
- **最小显示**: 12px 字号时清晰可见
- **推荐大小**: 16px - 32px
- **可缩放**: SVG 矢量格式，任意缩放不失真

## 最佳实践

1. **保持一致性**: 同一页面中使用相同大小的图标
2. **可访问性**: 为图标添加 `aria-label` 或 `title` 属性
3. **性能**: 按需导入图标，避免全量导入
4. **主题适配**: 使用主题变量控制颜色，支持亮/暗主题

```javascript
// ✅ 推荐：按需导入
import { HomeIcon, UserIcon } from '@yoya/basic';

// ❌ 不推荐：全量导入
import * as Icons from '@yoya/basic';

// ✅ 推荐：添加无障碍标签
const icon = HomeIcon({ 'aria-label': '首页', role: 'img' });

// ✅ 推荐：使用主题变量
icon.style('color', 'var(--yoya-text-primary)');
```
