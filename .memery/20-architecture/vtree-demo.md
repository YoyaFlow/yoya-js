# VTree 演示页面说明

**日期**: 2026-03-25
**文件**: `src/examples/yoya.tree.example.html`

---

## 演示页面结构

演示页面包含 4 个展示区域：

### 1. 基础树 (🌲 基础树)
- 支持节点展开/收起
- 支持节点选中
- 展示基础的树形结构

### 2. 复选框树 (✅ 复选框树)
- 支持级联勾选
- 支持半选状态
- 操作按钮：
  - 展开全部
  - 收起全部
  - 全选
  - 取消全选
- 实时显示当前勾选的节点 keys

### 3. 树形选择器 (📋 树形选择器)
- VTreeSelect 组件演示
- 下拉框 + 树形选择器
- 实时显示当前选中值

### 4. 大数据树 (📊 大数据树)
- 展示 1000 个节点的性能
- 操作按钮：
  - 加载 1000 节点
  - 恢复示例数据
- 演示虚拟元素架构的性能优势

---

## 事件日志

页面底部包含事件日志区域，显示所有用户交互事件：
- **select** 事件 - 绿色边框，节点选中
- **check** 事件 - 橙色边框，复选框勾选
- **expand** 事件 - 蓝色边框，节点展开/收起

支持清空日志功能。

## 访问方式

1. 启动开发服务器：
   ```bash
   npm run dev
   ```

2. 访问演示页面：
   - 直接访问：http://localhost:3004/examples/yoya.tree.example.html
   - 通过演示中心：http://localhost:3004/examples/yoya.example.html → VTree 树形组件
